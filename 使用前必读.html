准备工作

  • Node.js：确保你的电脑已安装 Node.js 或 Python。

操作步骤

1. 放置 3D 模型文件

将你的 3D 模型文件夹（例如包含 tileset.json 的文件夹）复制到 Tileset 目录中。

例如：

Tileset/你的模型文件夹名/tileset.json

2. 配置模型路径

如果你添加了新的模型，需要修改 index.html 中的 dataList 数组，将 unzipUrl 指向正确的模型文件。例如：

{
  dataName: "你的模型名称",
  type: 1,
  visible: false,
  unzipUrl: "./Tileset/你的模型文件夹名/tileset.json"
}

代码中已有示例，可参照修改。

3. 启动本地服务器（推荐）

为了解决无网络或无开发环境（Node.js/Python）的问题，我们提供了一个独立运行的服务器程序。

双击运行 server.exe

运行后：

  1. 会出现一个黑色命令行窗口（请不要关闭它）。
  2. 系统会自动打开默认浏览器并跳转到地图页面。
  3. 如果浏览器没有自动打开，请手动访问：http://localhost:8080

备选方案（如果有开发环境）

如果不想使用 exe，也可以使用以下方式启动：

Node.js:

npx http-server

Python:

# Python 3
python -m http.server 8080

4. 在浏览器中查看

打开浏览器，访问 http://localhost:8080（具体端口以终端显示为准）。

页面加载后，在左侧面板中找到你的模型，点击开关即可加载并显示 3D 模型。

如果模型未出现，可以按 Ctrl+R（Windows/Linux）或 Cmd+R（Mac）刷新页面重试。

注意事项

  • 模型文件夹内必须包含 tileset.json 文件，这是 Cesium 3D Tiles 的入口文件。
  • 如果模型较大，加载可能需要几秒钟，请耐心等待。
  • 保持终端窗口开启，关闭终端则服务停止。
